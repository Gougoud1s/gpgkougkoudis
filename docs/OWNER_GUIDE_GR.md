# Οδηγός Διαχείρισης — GP. ΓΚΟΥΓΚΟΥΔΗΣ

Αυτός ο οδηγός εξηγεί πώς να ενημερώνετε μόνοι σας τη νέα ιστοσελίδα, χωρίς να
χρειάζεται να καλέσετε προγραμματιστή. Όλες οι αλλαγές γίνονται μέσα από το
**Studio**, ένα φιλικό περιβάλλον σαν διαχειριστικό Wordpress.

## 1. Σύνδεση στο Studio

1. Ανοίξτε στον περιηγητή: `https://www.gpkougkoudis.gr/studio`
2. Πατήστε **Sign in with Google** και επιλέξτε τον λογαριασμό σας.
   - _Πρώτη φορά_: ζητήστε από τον προγραμματιστή να σας προσθέσει ως
     **Editor** στο project.
3. Θα δείτε αριστερά μια λίστα: Αρχική σελίδα, Ρυθμίσεις, Προϊόντα,
   Κατηγορίες, Υπηρεσίες, Κριτικές, FAQ, Άρθρα.

## 2. Πώς προσθέτω ένα νέο προϊόν

1. **Προϊόντα → + (κουμπί νέο πάνω δεξιά)**
2. Συμπληρώστε:
   - **Τίτλος (Ελληνικά)** και **Τίτλος (English)** — π.χ. `Μονόπετρο Aurora` /
     `Aurora Solitaire`
   - **URL slug** — πατήστε _Generate_ για αυτόματη δημιουργία (μπορείτε
     να το πειράξετε π.χ. `monopetro-aurora`)
   - **Κωδικός (SKU)** — αν έχετε
   - **Κατηγορία** — επιλέξτε από τη λίστα (π.χ. _Δαχτυλίδια_)
   - **Φωτογραφίες**:
     - Σύρετε φωτογραφίες με drag & drop μέσα στο πλαίσιο
     - Ανεβάζετε όσες θέλετε ταυτόχρονα
     - Για να αλλάξετε τη σειρά, πιάστε με το ποντίκι
     - Για να ορίσετε σημείο εστίασης (κοντινό), κάντε διπλό κλικ στη
       φωτογραφία και τοποθετήστε το σημείο _hotspot_
     - Καλό είναι κάθε φωτογραφία να έχει **alt text** (περιγραφή για
       τυφλούς και Google) — π.χ. `Χρυσό μονόπετρο 18Κ`
   - **Υλικό, Καράτια, Πέτρα, Περίσταση** — επιλέξτε από λίστες
   - **Τιμή (€)** — γράψτε αριθμό· αν αφήσετε κενό ή ενεργοποιήσετε
     **Τιμή κατόπιν αιτήματος**, η σελίδα θα δείχνει "Τιμή κατόπιν
     αιτήματος"
   - **Περιγραφή** — Ελληνικά + Αγγλικά
   - **Διαθέσιμο στο κατάστημα** — αφήστε ON
   - **Προβολή στην Αρχική** — ON για να εμφανιστεί στην homepage
3. Πάνω δεξιά πατήστε **Publish** (κίτρινο κουμπί).
4. Ανοίξτε τη σελίδα να δείτε το αποτέλεσμα.

## 3. Πώς αλλάζω/διαγράφω φωτογραφίες ενός προϊόντος

1. Ανοίξτε **Προϊόντα → επιλέξτε το προϊόν**.
2. Στο πεδίο **Φωτογραφίες**:
   - **Διαγραφή**: κάντε κλικ στη φωτογραφία → πατήστε το εικονίδιο _κάδος_
     ή το _X_.
   - **Αντικατάσταση**: κάντε κλικ → _Replace_ → ανεβάστε νέα φωτογραφία.
   - **Σειρά**: σύρετε με το ποντίκι.
3. **Publish**.

## 4. Πώς αλλάζω την Αρχική σελίδα

1. **Αρχική σελίδα** (πρώτο στοιχείο στη λίστα).
2. Μπορείτε να αλλάξετε:
   - **Hero — εικόνα** (μεγάλη φωτογραφία στην κορυφή)
   - **Hero — τίτλος / υπότιτλος** (Ελληνικά + Αγγλικά)
   - **Επιλεγμένες κατηγορίες** (4 το μέγιστο)
   - **Επιλεγμένα προϊόντα** (8 το μέγιστο)
   - **Επιλεγμένες υπηρεσίες** (3 το μέγιστο)
   - **Επιλεγμένες κριτικές** (6 το μέγιστο)
   - **Story** (τίτλος + κείμενο + φωτογραφία)
3. **Publish**.

## 5. Πώς ενημερώνω ωράριο, τηλέφωνο, social media

**Ρυθμίσεις ιστοσελίδας → ενημερώστε** → **Publish**.
Οι αλλαγές εμφανίζονται παντού (header, footer, contact page) αυτόματα.

## 6. Πώς προσθέτω μια νέα κριτική

1. **Κριτικές → +**
2. Όνομα πελάτη, αστέρια (1–5), κείμενο (Ελληνικά + Αγγλικά μεταφρασμένα),
   πηγή (συνήθως _Google_), ημερομηνία.
3. Ενεργοποιήστε **Προβολή στην Αρχική** για να εμφανιστεί.
4. **Publish**.

## 7. Πώς προσθέτω/αλλάζω συχνή ερώτηση

**FAQ → +** → ερώτηση + απάντηση (EL/EN) → επιλέξτε κατηγορία → **Publish**.

## 8. Συμβουλές για όμορφες φωτογραφίες προϊόντων

- Φωτογραφίστε με **σταθερή φωτογραφική μηχανή** ή κινητό
- Χρησιμοποιήστε **σκούρο φόντο** (μαύρο βελούδο ή σκούρο χαρτί)
- Φως **ομοιόμορφο** — αποφύγετε σκληρές σκιές
- 4–6 φωτογραφίες ανά προϊόν: μια σύνθεση, μια κοντινή στην πέτρα, μια
  πλάγια, μια φορεμένο
- Συνιστάται **τετράγωνη αναλογία** (1:1) και τουλάχιστον 1200×1200 pixels

## 9. Συντομεύσεις πληκτρολογίου

| Ενέργεια        | Συντόμευση       |
| --------------- | ---------------- |
| Αποθήκευση      | `Cmd/Ctrl + S`   |
| Δημοσίευση      | `Cmd/Ctrl + Alt + P` |
| Αναζήτηση       | `Cmd/Ctrl + K`   |

## 10. Δεν λειτουργεί κάτι;

- Πατήστε **Publish** ξανά
- Ανανεώστε τη σελίδα της ιστοσελίδας (`Cmd/Ctrl + Shift + R`)
- Στείλτε email στον προγραμματιστή

---

**Βίντεο οδηγός** (≈5 λεπτά): θα βρείτε τον σύνδεσμο στο email μετά την
παράδοση, καθώς και αυτόν τον οδηγό σε εκτυπώσιμο PDF.
